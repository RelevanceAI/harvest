# Plans Directory

Plans are organized here by branch name with timestamped files.

**For full planning guidelines, workflow, and organization:**

See `@docs/ai/shared/planning.md`

## Quick Reference

```
.claude/plans/
└── [branch-name]/
    ├── research_YYYY-MM-DD_HHMM.md
    ├── plan_YYYY-MM-DD_HHMM.md
    └── implementation_YYYY-MM-DD_HHMM.md
```

All planning process documentation lives in the shared rules, not here.
